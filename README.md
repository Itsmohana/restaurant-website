A responsive multi-page restaurant website built using HTML5, CSS3, and JavaScript.

## Overview
This project is a frontend practice project focused on building a structured and responsive restaurant website.  
The goal was to practice layout design, page structuring, and basic interactivity.

## Features
- Responsive layout design
- Multi-page navigation (Home, Menu, About, Contact)
- Clean and structured UI
- Basic JavaScript interactions

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
This project was developed as part of my frontend learning journey to strengthen my understanding of web structure, styling, and responsive design principles.
